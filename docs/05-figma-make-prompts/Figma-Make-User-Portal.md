# Figma Make Prompts: User Portal

## Scope

User portal for daily operations, personal settings, workflow tasks, notifications, and account visibility.

## Prompt Set

### Prompt 1: User Dashboard Experience

```text
Design a user dashboard with:
- prioritized task list
- recent activity timeline
- notification digest
- quick actions panel
Provide desktop and mobile variants with clear hierarchy.
```

### Prompt 2: Onboarding and First-Run

```text
Create first-run experience:
- welcome screen
- profile completion
- permission and context explanation
- guided product tour
- completion confirmation
```

### Prompt 3: Workflow Task Screens

```text
Design task execution screens:
- list view with filters
- detail view with comments and history
- submit/update actions
- success and failure states
```

### Prompt 4: Notifications and Preferences

```text
Build notification center and settings:
- inbox view with severity tags
- channel preferences (email, in-app, webhook where applicable)
- do-not-disturb scheduling
- retry/help states for failed delivery
```

### Prompt 5: Billing Visibility for Standard Users

```text
Create read-only billing visibility components:
- plan badge
- usage widget
- invoice access card
- support escalation CTA
```

### Prompt 6: Microcopy and Interaction

```text
Generate microcopy for empty states, validation errors, and guidance tips.
Define motion behavior for page transitions, drawers, and toasts with timing values.
```

### Prompt 7: Responsive and Accessibility Requirements

```text
Ensure all user flows are responsive and accessible:
- keyboard navigation map
- focus indicators
- color contrast annotations
- screen reader label recommendations
```
