# UX Strategy

## Experience Target
ERP-eCommerce must provide category-leading speed, clarity, and decision support.

## Role-Centric Journeys
- Guided onboarding by role
- Context-preserving deep links between records and modules
- Keyboard and command palette parity for power users

## UI Quality Bars
- Lighthouse Performance >= 95
- Lighthouse Accessibility >= 95
- End-to-end critical path completion < 3 interactions for common tasks

## Interaction Patterns
- Optimistic UI for safe mutations
- Autosave for long forms
- Inline AI recommendations with confidence and rationale
- Unified notification center with priority ranking
