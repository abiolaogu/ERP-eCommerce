# Training Manual

## Program Overview

This training program enables Product, Admin, and operational teams to use and operate the System safely and effectively.

## Tracks

- **Track A**: End User operations
- **Track B**: Tenant and Org administration
- **Track C**: Support and operations response

## Session Plan

| Session | Audience | Duration | Outcomes |
|---|---|---|---|
| Product overview | All roles | 45 min | Understand Product, Tenant, Org model |
| User workflows | End Users | 60 min | Complete core tasks and notifications |
| Admin workflows | Admins | 90 min | Manage Users, roles, billing, integrations |
| Security and compliance | Admins + Ops | 60 min | Apply policy and audit best practices |
| Incident simulation | Ops + Admins | 75 min | Execute runbooks under timed conditions |

## Practical Exercises

1. Create a Tenant and invite an Org Admin.
2. Assign and revoke a role with audit verification.
3. Trigger and review a billing charge event.
4. Simulate failed webhook delivery and recovery.

## Assessment Criteria

- Correctly executes role assignment with policy constraints.
- Demonstrates audit event retrieval for a given incident.
- Completes rollback simulation using documented procedure.

## Enablement Assets

- End user quick-reference card
- Admin operation checklist
- Incident response call tree template

## Ongoing Learning

- Monthly refresher for Admin and Ops cohorts.
- Quarterly tabletop exercise for incident and DR readiness.
