# End User Guide

## Audience

End Users who perform day-to-day tasks in the Product web portal.

## Getting Started

1. Sign in with your User account.
2. Select your Org and working context.
3. Use dashboard tiles to access assigned workflows.

## Core Workflows

### 1) View Dashboard

- Navigate to **Home > Dashboard**.
- Review assigned tasks, recent events, and notifications.
- Use date and status filters to focus your work.

### 2) Manage Personal Profile

- Open **Settings > Profile**.
- Update display name and notification preferences.
- Save and confirm profile update notification.

### 3) Track Task Status

- Open **Workflows > My Tasks**.
- Filter by status (`pending`, `in_progress`, `completed`).
- Open task details and submit updates.

### 4) Review Notification Inbox

- Open **Notifications**.
- Filter by severity and channel.
- Mark items as read or escalate to Admin.

## UX Notes for Documentation Screenshots

For internal documentation screenshots, capture:

- dashboard with task widget and notification panel
- profile settings form with validation messages
- task details drawer and action timeline
- notification center with filters

## Accessibility and Usability

- Keyboard navigation supported for all primary actions.
- Color contrast meets WCAG AA baseline.
- Validation messages are clear and actionable.

## Common User Issues

- Session timeout: refresh and sign in again.
- Missing permissions: contact Org Admin.
- Delayed notifications: check channel settings and status page.
