## Data Room Access Protocol

1. All documents watermarked with recipient fund name and access date
2. View-only by default; download enabled only for P0 documents after signed NDA
3. Access logged with IP and timestamp; reviewed weekly by CEO
4. Data room expires 90 days after fundraise close
5. Post-close: transition to quarterly investor update cadence
