# Security Policy

## Supported Versions

This documentation repository supports the latest `main` branch for security updates and policy corrections.

## Reporting a Vulnerability

If you discover a security issue affecting documented controls, architecture guidance, or release procedures:

1. Email: `security@product.example`
2. Include:
   - summary of issue
   - potential impact
   - affected document path(s)
   - recommended mitigation
3. Do not disclose publicly until triage is complete.

## Response Targets

- Initial acknowledgement: within 1 business day
- Triage and severity assignment: within 3 business days
- Remediation plan: within 5 business days

## Security Contacts

- Security Lead: `security-lead@product.example`
- Platform Security: `platform-security@product.example`
- Compliance Lead: `compliance@product.example`

## Disclosure Policy

After mitigation is merged and validated, we publish a summarized advisory in release notes.
